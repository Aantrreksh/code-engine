# Running jobs with Lithops framework
{: #lithops} 

Python users can benefit of using Lithops framework to integrate with Code Engine. 
Lithops enables native Python integration with Code Engine by using Lithops API. Lithops then transparently to the user
scale the code and execute it against Code Engine at massive scale. All this happens without CLI tools or building yaml descriptions.

## Running your first flow with Lithops framework
{: #first-lithops} 

1. Navigate to https://github.com/lithops-cloud/lithops and install Lithops
2. Follow https://github.com/lithops-cloud/lithops#quick-start and setup IBM Code Engine and IBM COS
3. Run "hello world" example:

```
import lithops

iterdata = [1,2,3,4,5]

def my_map_function(data):
    return data + 1

if __name__ == '__main__':
    pw = lithops.function_executor(type = "serverless", backend='code_engine') 
    futures = pw.map(my_map_function, iterdata)
    print (pw.get_result())
    pw.clean()
```
